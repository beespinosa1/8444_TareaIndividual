Qué es?

JavaScript Asíncrono + XML (AJAX) no es una tecnología por sí misma, es un término que describe un nuevo modo de utilizar conjuntamente varias tecnologías existentes. Permite realizar aplicaciones web capaces de actualizarse continuamente sin tener que volver a cargar la página completa.

Composición

  • XHTML y CSS para presentar información
  • DOM para visualizar e interactuar de forma dinámica la información presentada
  • XMLHttpRequest para manipular los datos de forma asíncrona con el servidor web
  • XML, HTML y XSLT para el intercambio y la manipulación de datos.

Caracteríticas

La característica más atractiva de AJAX es su naturaleza "asincrónica", lo que significa que puede comunicarse con el servidor, intercambiar datos y actualizar la página sin tener que actualizar la página.
